# hft-widget 

Simple high-frequency trading educational widget using [limit-order-book](https://github.com/rhodey/limit-order-book).

## Setup
```
$ git clone https://github.com/rhodey/hft-widget
$ cd hft-widget
$ npm install
```

## Bundle
```
$ npm run bundle
```

## Testing
```
$ npm run dev
```
